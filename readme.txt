== Installation ==
1. Install woocommerce plugin
2. Upload `Taobao Catalog` folder to the `/wp-content/plugins/` directory
3. Register on  http://taobaowoocommerce.dropshop.pro/, use <a href="http://taobaowoocommerce.dropshop.pro/wp-login.php?action=register">registration link</a>   
4. Use your e-mail and password, after activation your account on http://taobaowoocommerce.dropshop.pro/  in Taobao Catalog Settings
